## 📂 Projects  

- 🧭 **Poliglota** – AI-powered language learning assistant ([poliglota.ai](https://www.poliglota.ai))  
  I’m currently building **Poliglota**, a private, research-oriented platform designed to support autodidacts in learning new languages through AI and literature.  
  - It features a **Retrieval-Augmented Generation (RAG)** system built on classic literature, philosophy, and non-fiction, to generate meaningful comprehension exercises grounded in real texts.  
  - A real-time **oral tutor** simulates immersive speaking sessions via GPT-based APIs, adapting to user level and conversation themes.  
  - A custom **browser extension** allows users to select text anywhere online and generate intelligent flashcards, complete with context-aware explanations, grammar tips, and reformulations.  
  - Poliglota explores the intersection of **language, cognition, AI**, and **self-directed learning**, with an emphasis on depth, nuance, and long-term memory formation.

- 🎙️ **Bobby – LLM-Supervised VoiceBot for Real-Time Call Center Monitoring**  
  Bobby is a voice-based conversational agent developed for **real-time supervision** in call centers at IMA. It enables supervisors to **query operational metrics** (DMR, transfers, abandonment rate, etc.) using natural language, without interacting directly with Kibana elastic search database.  
  - Natural voice interaction via Web Speech API, with GPT-based intent parsing and context handling  
  - Real-time data retrieval from Elasticsearch through Kibana’s API, and spoken output via Google Cloud TTS  
  - Intelligent **query mapping and summarization** using LLMs, with adaptive reformulations  
  - Hardware-triggered alerts through microcontroller when predefined thresholds (e.g., DMR) are exceeded  
  - Full-stack implementation using Python (FastAPI), OpenAI API, Elasticsearch, and Google Cloud TTS  

🛰️ Drone Vision Prototype – Real-time image capture & remote detection on Raspberry Pi
I’m developing a lightweight drone vision prototype using a Raspberry Pi and USB camera. It captures 1 image per second for 60 seconds, tags each file with GPS (via IP) and timestamp, then sends it to a remote AI worker that detects green objects.
Each result is logged (filename, coords, timestamp, detection), and the system was tested successfully by filming indoor plants. It lays the foundation for onboard visual intelligence in future drone applications.
